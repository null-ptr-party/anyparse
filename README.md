# anyparse
Library used for generic message parsing.
## Installation
1. Install Msys from the link here: https://www.msys2.org/
2. Open Msys UCRT64.
3. Update packages using command pacman -Syuu
4. Install the necessary compilation tools using pacman -S mingw-w64-ucrt-x86_64-toolchain
5. compile the dll using make
